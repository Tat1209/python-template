# Python Library Template

本リポジトリは、パッケージマネージャーに `uv`、Linterおよびフォーマッタに `Ruff` を採用した汎用ライブラリ開発用のテンプレートです。

コードの静的解析とフォーマット処理を完全に `Ruff` へ統合しているため、エディタ（VSCode等）環境における `Pylance` の一部機能（重複するLint機能や警告など）は、処理の競合や二重警告を回避する目的で無効化される前提で構成されています。型の解決のみを `Pylance` に委譲します。

## 環境構築手順

本テンプレートは，特定のPythonマイナーバージョンをGit上で強制しません。利用者は以下の手順に従い、ローカルで開発に使用するPythonバージョンを明示的に指定したうえで環境を構築してください。

```bash
# 1. 開発に使用するPythonバージョンを指定（ローカル専用の .python-version を生成）
uv python pin 3.12

# 2. 依存関係の同期および仮想環境（.venv）の構築
uv sync
```
